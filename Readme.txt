About Operating
关于操作

    First install the modules we need (listed at bottom)
    首先要安装必要的库（在末尾列出）

    No Chinese in absolute path
    绝对路径中不能出现中文

    Check the MD5 Value of "main.py": C4E1C86E038443D5171C0B4B0FE319C6
    校验“main.py”的MD5值：C4E1C86E038443D5171C0B4B0FE319C6

    1. Use the terminal/command prompt to call Python(3) to run the file "main.py"
         Call the camera (Please make sure your computer has a camera)
         Open the window named "Green Hat !" to display the video shot in real-time
         Draw a green hat photo on top of the recognized face
    2. Focus on the video window and press Esc to exit; or exit by closing terminal/command prompt

    1. 使用终端/命令提示符调用Python（3）执行“main.py”文件
        调用摄像头（请确保你的计算机有摄像头）
        打开名为“Green Hat !”的窗口显示实时拍摄的视频
        将绿帽照片绘制到所识别到的人脸的头顶
    2. 此时将焦点移动至视频窗口，按下Esc键即可退出；也可通过关闭终端/命令提示符退出

    You can see an example screenshot named "example.jpg" in the project folder
    项目文件夹里有名为“example.jpg”的示例图
